Research Build · 2026

LLM Code Graph Compiler

Deterministic Workflow-to-Code

Separates LLM planning from execution entirely. The model selects nodes from a pre-verified typed registry and outputs a JSON plan. A seven-check static validator enforces correctness before any code runs; no repair loops, no runtime chaining, no silent failures. The evaluation surfaced a "constraint evasion" failure mode: leave one surface unconstrained and the planner always finds it. That turned out to be the most interesting result.

SVMP — Multi-Tenant LLM Governance Layer

Multi-tenant governance architecture for customer service AI workflows. Three async workflows; ingestor, processor, janitor, each doing one job. A 2.5s soft debounce and atomic lock queue merge fragmented messages and cut duplicate LLM calls by ~40%. Tenant isolation via a four point identity frame. 0.75 cosine-similarity gate routes uncertain queries to humans with Slack escalation. API keys encrypted at rest with Fernet AES-128. Ran 500+ adversarial tests for prompt injection and cross-tenant leakage.
