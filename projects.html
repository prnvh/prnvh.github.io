Lucid Model

Artificial-brain prototype using EBMs, Hopfield-style attractor memory, lazy collapse, binding, interference, projection, and lucidity.

The goal is inspectable reasoning and cheaper continual learning than weight-only models. Memory stays explicit; the system commits only when its state is stable enough.

Proof: repo, constitution notes, audit/checkpoint machinery, CLI stage coverage, and 212 pytest tests in the training suite.

Memory Infrastructure · 2026

Wiki Memory / RAW SOURCES

Long-running memory layer that turns immutable raw sources into canonical wiki pages, relation graphs, temporal facts, and answer-ready context.

Built so agents can recall sourced, revisable memory instead of replaying transcripts.

  • LongMemEval-S wiki-index run: 470 cases, hit@1 88.7%, hit@5 97.9%, hit@10 99.1%, MRR 0.924.
  • Temporal and aggregation paths compute date/order/count/sum/list support in code where possible.
