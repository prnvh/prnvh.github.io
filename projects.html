Research Build · 2025-2026

LLM Code Graph Compiler

Compile structured workflows into executable code through a typed node registry and static validation, rather than letting the model improvise its way through runtime.

Python DAG Execution Static Validation CLI Tooling

What it is

The model only selects from pre-verified nodes and emits a JSON plan. A validator checks type correctness, wiring, argument structure, and graph integrity before any execution begins.

Why it mattered

The interesting result was not just higher compile success. It was discovering how quickly planners exploit any unconstrained surface, which sharpened how I think about system boundaries.

What I focused on

  • Typed registry design and deterministic code generation
  • Static safety checks ahead of runtime
  • Benchmarking against freer-generation baselines

Research Build · 2026

Governed Memory Layer for Multi-Agent Shared State

A memory runtime where agents write raw notes privately, then promote only validated and resolvable state into canonical shared memory through a ledger-backed pipeline.

Python SQLite Shared Memory Deterministic Projection

What it is

The core split is between working memory and canonical memory. Notes are interpreted, resolved, validated, and only then written through a single official path into events, projections, and pending state.

Why it mattered

The hard problem turned out to be identity continuity rather than storage. That pushed the design toward pending ambiguity, reference memory, and a stronger refusal to guess.

What I focused on

  • Governed promotion pipeline and narrow write boundaries
  • Append-only event durability with deterministic projection
  • Pending resolution for lifecycle updates that cannot safely bind

Systems Build · 2026

SVMP Multi-Tenant Governance Layer

A customer-service reliability layer for multi-tenant AI workflows, with queue discipline, tenant isolation, escalation routing, and operational guardrails.

What it is

Three async workflows handle ingestion, processing, and cleanup. A soft debounce plus lock queue merges fragmented messages before the LLM layer sees them, which reduces duplicate work and sharpens session state.

Why it mattered

This was where operational reliability stopped being abstract. Isolation boundaries, uncertain-query routing, and adversarial testing all had to work as product constraints rather than nice-to-have theory.

Field Research · 2026

Wage Asymmetry Study in Kerala Labour Markets

Structured interview-based research on the wage gap between inter-state migrant workers and local labourers in Kerala.

What I did

Designed the interview structure, ran 150-plus field interviews, and started shaping the findings into a paper rather than leaving them as raw anecdote.

Why it mattered

This work sharpened how I think about labour markets, asymmetric information, and the difference between formal system design and the real-world behaviour around it.
