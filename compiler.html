PlanCompiler

A typed compiler for LLM workflow graphs.

PlanCompiler asks the model for a structured JSON plan, validates the plan against a node registry, then generates executable code deterministically. The model chooses the graph; the compiler handles the execution surface.

The point was to reduce multi-step workflow failure. Free-form code generation often breaks as tasks get deeper: imports drift, schemas mismatch, column names change, and repair loops hide the original error. PlanCompiler moves the unstable part earlier and keeps the later path typed.

01 300 benchmark tasks across six workflow sets.

02 278/300 compiled successes, compared with 202/300 for GPT-4.1 free-form Python in the same harness.

03 Seven static checks before execution: node existence, edge validity, type compatibility, acyclicity, orphan detection, input arity, and required parameters.

04 Proof: arXiv paper / GitHub repo.
